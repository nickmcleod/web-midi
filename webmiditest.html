Web MIDI API test

This page calls the new Web MIDI API to detect MIDI devices attached to your computer.

As of May 2015, this API has been implemented in the latest versions of the Chrome and Opera browsers - see the latest browser support list.

Notes:
  • Your browser may prompt you for permission to run scripts or access the MIDI API.
  • Some versions of Chrome may need to have the "Enable Web MIDI API" enabled on the settings page:
    1. Enter this in the Chrome address bar: chrome://flags.
    2. Search in the page for 'midi'.
    3. If the setting "Enable Web MIDI API" exists, click the 'Enable' link just below it.
      If the link says 'Disable', the API has already been enabled.
      If the setting doesn't exist, you either have a very old version of Chrome, or the latest version which doesn't need this setting as it is enabled by default.
